사용 모델: Gemini

다음의 기능을 하는 로컬에서 실행할 수 있는 Python 코드를 작성해 줘.
지정된 이름의 UTF-8 인코딩의 txt 파일을 읽어옴. (이 파일을 작업파일이라 부르자) 이 파일은 1개 열로 된 데이터임.
또 다른 파일인 guangyun.txt를 읽어옴. 이 파일은 TSV 형식으로 된 여러 열의 데이터임.
사용자에게 guangyun.txt의 열 이름을 입력받음(그 열을 "지정 열"이라 부르자)
작업파일에 "지정 열"과 같은 이름의 새로운 열을 생성.(제2열이 됨)
작업파일을 한 줄 씩 읽음. 그 줄의 문자열과 일치하는 셀이 guangyun.txt의 "字" 열에 있으면 해당되는 행의 지정 열 값을 작업파일의 제2열에 입력. 만약 해당되는 행이 여럿이면 입력해야 하는 값들을 "/"로 구분하여 하나의 셀에 넣음.
모든 작업파일의 행에 대하여 작업을 마치면 원래의 파일 이름 뒤에 지정 열의 이름을 붙여서 새로운 파일로 저장.

기능을 하나 추가해 줘. 매칭되는 글자가 없으면 사용자에게 관련 정보를 알리고 대신 찾을 글자를 입력받아서 그 글자를 기준으로 매칭을 수행. 그래도 매칭되는 글자가 없거나, 사용자가 아무 것도 입력하지 않았으면 빈칸으로 둠.

새로운 기능 추가. 이제 사용자가 입력해 주는 데이터에 기반하여 이형자 라이브러리 파일을 만듦. 매칭되지 않은 글자가 발견되면 이형자 라이브러리를 먼저 참조하여 매칭하고, 라이브러리에 없을 때에는 원래대로 사용자에게 물음. 이형자 라이브러리는 프로그램 중단/종료시에도 사라지지 않도록 폴더에 저장하고, 새로 실행할 때에도 이 라이브러리를 활용.

기능 수정. 이제 이형자 라이브러리가 업데이트될 때마다 저장하고 그 내용을 다시 불러옴.
